# Infiniti Servizi — Struttura Organizzativa

Documenti organizzativi per Infiniti Servizi S.r.l.

## Contenuto

- **index.html** — Pagina di navigazione
- **organigramma-mansionario.html** — Organigramma target, mansionario per ruolo, procedure operative, governance transitoria
- **matrice-assegnazione.html** — Matrice interattiva persona → ruolo (con dropdown, aggiunta/rimozione dipendenti, salvataggio locale)
- **assistente-presidenza.html** — Mansionario ruolo Assistente di Presidenza

## Deploy

Questo repo è pensato per GitHub Pages. Attivalo da Settings → Pages → Source: main branch, root (/).

## Note

I dati della matrice di assegnazione vengono salvati nel localStorage del browser. Non sono condivisi tra dispositivi.
